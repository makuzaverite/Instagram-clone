## iDrip

> MERN project inspired by instagram

#### what is does

- Loggin user
- Registering user
- Adding post
- Deleting a post
- Edit profile

### How use use it on your machine

```git
  git clone https://github.com/makuzaverite/iDrip.git

  cd iDrip

  npm install

  cd client

  npm install
```
